# Superior Roofing & Home Solutions

Static multi-page website prepared for Vercel.

## Pages
- `/` home page
- `/about` about and service-area page
- `/financial` payment page
- `/blog` blog placeholder page

## Deploy to Vercel
1. Upload this folder to a GitHub repo.
2. Import that repo into Vercel.
3. Vercel will deploy it as a static site automatically.

## Important placeholders to replace
- Payment link on `financial.html`
- Social profile links on all pages
- Any brand assets or photos you want to add later
